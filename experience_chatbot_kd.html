✨ Golden Tree Chatbot Showcase

📌 How It Works:

  • ✅ Auto greeting: Bot greets and shows numbered options.
  • 📝 User Input: Type or click “Send” or press Enter.
  • 🤖 Bot Replies: Based on text or option selected (e.g., address, timings).
  • 📍 Map Integration: Ask for location, click "Show in Map" to open Google Map modal.
  • 🗑️ Delete Feature: Trash icon opens a confirmation modal to clear messages (with localStorage).
  • 📂 Storage: Conversations are stored and auto-loaded on refresh.
  • Greeting Automation: Welcomes users with clear option buttons.
  • Input Flexibility: Users can type or use option buttons for fast replies.
  • Delete Confirmation: Clicking the trash icon shows a confirm modal before clearing data.
  • Google Maps Integration: Clicking “Show in Map” opens location modal with embedded map.
  • Mobile Responsive: Tailwind CSS ensures smooth UX on all screen sizes.
  • Clean UI with Icons: Uses intuitive icons for actions (delete, send, etc.).

Note: You can expand this with WhatsApp integration, product carousel, or AI assistant too.
